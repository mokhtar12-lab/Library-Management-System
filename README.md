# Library-Management System ( Java + JDBC + MySQL )
A backend console-based Application build with Java applying core concept like OOP, Collections, Exception Handling, and JDBC for database connectivity

# Features 
  . Manage Books & Users 
  . Borrow & return System
  . Full CRUD Operations ( Create, Read, Update, Delete )
  . MySQL Database Integration using JDBC 
  . Clean Code Structure With DAO Pattern

# Tech Stack :
Java | MySQL | JDBC | Maven 

This Project demonstrates solid understanding of backend fundamentals and database integration.
